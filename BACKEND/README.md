# BACKEND

🧠 Vibe Check Backend
Это серверная часть приложения для трекинга настроения, написанная на языке Go.

🛠 Технологии
Язык: Go (Golang)

База данных: MySQL / MariaDB

Роутинг: Стандартная библиотека net/http с поддержкой CORS

📂 Структура проекта
api/v1/ — хендлеры (обработчики) для всех запросов API

cmd/server/main.go — точка входа в приложение, настройка маршрутов и базы

internal/models/ — структуры данных (MoodLog, User и т.д.)

internal/repository/ — логика работы с базой данных (SQL-запросы)

go.mod — управление зависимостями проекта

🚀 Запуск проекта
Настройка базы данных:

Импортируйте SQL-дамп в вашу БД.

Убедитесь, что БД доступна по адресу, указанному в main.go. По умолчанию: root:rootroot@tcp(127.0.0.1:3307)/mood_tracker.

Запуск сервера:

Bash
cd BACKEND
go run cmd/server/main.go
Сервер запустится на порту :8080.

🔗 Основные API Endpoints
POST /api/register — Регистрация нового пользователя

POST /api/login — Авторизация

GET /api/logs — Получение истории записей настроения

POST /api/logs/save — Создание новой записи

GET /api/stats — Статистика настроения

GET /api/ping — Проверка доступности сервера
